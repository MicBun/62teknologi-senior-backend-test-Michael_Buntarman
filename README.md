## How to run
### Docker Compose
```
1. docker-compose build
2. docker-compose up
```

### Docker
```
1. docker build -t go-microservice-kubernetes .
2. docker run -p 8080:8080 go-microservice-kubernetes
```

### Makefile
```
1. make run
```

### test
```
1. make test
```

### Coverage
```
1. make coverage
```
